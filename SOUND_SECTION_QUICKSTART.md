# Sound Section - Quick Start

## ⚡ Setup Cepat (2 Menit)

### 1. Import Database
```bash
psql -U postgres -d landing_cms -f sound_section_setup.sql
```

### 2. Refresh Browser
Buka: http://localhost/lending_word/

### 3. Edit Konten (Optional)
1. Login admin: http://localhost/lending_word/admin/
2. Klik tab **"Sound"**
3. Edit title, caption, image, button text
4. Save Changes

## ✅ Selesai!

Sound section sudah aktif di landing page dengan style cinematic seperti referensi Porsche.

## 📸 Preview

Section akan muncul setelah "Explore Models" dengan:
- Full-screen background image
- Title besar di tengah
- Caption/subtitle
- Button "Hold for sound" dengan icon play

## 🎨 Default Content

- **Title**: "Set the pace: 9,000 revolutions per minute."
- **Caption**: "The naturally aspirated engine and sport exhaust system ensure an unfiltered sound experience."
- **Image**: Porsche GT3 RS
- **Button**: "Hold for sound"

Semua bisa diubah via admin panel!
