## рҹ–јпёҸ Image Requirements

### Background Image
```
Recommended:
- Size: 1920x1080px (Full HD)
- Format: JPG/PNG
- Quality: High (80-90%)
- Aspect: 16:9
- Subject: Car in environment
```

### Carousel Card Images
```
Recommended:
- Size: 800x600px
- Format: JPG/PNG
- Quality: Medium-High (70-80%)
- Aspect: 4:3
- Subject: Detail shots (engine, interior, etc)
```

## рҹ”„ Carousel Logic

```javascript
Current Index: 0

User clicks [вҶ’]:
в”ңв”Җв”Җ Index + 1
в”ңв”Җв”Җ If Index >= Total Cards: Index = 0
в””в”Җв”Җ Translate: -Index Г— CardWidth

User clicks [вҶҗ]:
в”ңв”Җв”Җ Index - 1
в”ңв”Җв”Җ If Index < 0: Index = Total Cards - 1
в””в”Җв”Җ Translate: -Index Г— CardWidth
```

## рҹҺ¬ Animation Flow

```
Page Load:
1. Background image loads (parallax)
2. Overlay fades in
3. Text appears
4. Cards slide in from right
5. Arrows fade in

User Interaction:
1. Click arrow
2. Track translates (0.5s ease)
3. Cards slide smoothly
4. New cards visible
```
